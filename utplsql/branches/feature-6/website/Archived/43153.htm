Announcing Web Unit Testing PLSQL (WUTPLSQL).

Attached is the beginnings of a Web Interface for
UTPLSQL that I have been working on. Right now it
only performs some basic functions :

1) Define and test suites
2) Test packages
3) Generate test packages

I am slowly adding functionality as I learn HTML and
specifically the PL/SQL Web Toolkit which is required
to run WUTPLSQL. It also requires a PL/SQL Gateway which
is part of OAS, iAS, WebDB(?) or the HTTP Server
in 8.1.7.

A simple user manual is included in the attached
zip.
